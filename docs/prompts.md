# Prompts

## Vision Document

Write a product vision document for an application called **AICORD** — a lightweight, AI-native messaging platform inspired by Discord. The document should cover:

- **Overview**: What AICORD is, its core purpose, and how it differs from Discord
- **Target Users**: Who it is built for and the problems it solves
- **Key Features**:
  - Multi-user support (accounts, authentication, presence)
  - Servers and channels (text, voice, or AI-specific channel types)
  - AI-powered capabilities (e.g., AI assistants per channel, smart summarization, context-aware replies)
  - Real-time messaging and notifications
  - Roles and permissions
- **How It Works**: A high-level technical and UX walkthrough of a typical user session
- **AI Integration**: How AI is woven into the product — not bolted on, but core to the experience
- **Design Principles**: The values that guide product decisions (simplicity, intelligence, openness, etc.)

Write in a clear, professional tone suitable for a product spec or investor brief. Use headers, bullet points, and short paragraphs for readability.
